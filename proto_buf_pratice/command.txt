

1. **Basic compile inside `proto/` folder**

```bash
cd proto
protoc --go_out=. main.proto
cd ..
```

2. **Compile from root directory with path**

```bash
protoc --go_out=. ./proto/main.proto
```

3. **When you have nested `.proto` files (e.g., `user/user.proto`, `order.proto`)**

```bash
protoc -I=proto --go_out=. proto/main.proto proto/user/user.proto proto/order.proto
```

* `-I=proto` tells `protoc` where to look for imports.
* This is the **standard way** if you have multiple `.proto` files.
